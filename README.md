Relatively simple Linux config for desktop, managed with [yadm](https://github.com/TheLocehiliosan/yadm).

- OS: [Void Linux][void]
- window manager: [spectrwm][spectrwm]
- shell: [fish][fish]
- editor: [neovim][neovim]
- menu: [bemenu][bemenu]
- hotkey daemon: [sxhkd][sxhkd]


[void]: https://voidlinux.org
